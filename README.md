# twabot
See Twitch Chat's mood at a glance.

A project by Lukas Edelböck, Felix Kampfer, and Andreas Wundlechner

The project is devided in 3 parts:
1. Fetching the data from Twitch and storage in Rethinkdb
	Provided by Lukas Edelböck
1. Aanalysing the mood of the chats
	Provided by Felix Kampfer
1. Displaying the analysed data with a webfrontend
	Provided by Andreas Wundlechner
